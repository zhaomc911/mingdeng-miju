# Privacy Policy

Last updated: 2026-04-25

Mingdeng Miju (明灯迷局) is currently a client-side puzzle game. This document describes the current privacy behavior of the project.

## Data We Store

The current web version may store game progress locally in the user's browser, for example unlocked levels or local play state.

This local data stays on the user's own device and is not sent to a project backend by the app.

## Data We Do Not Collect

The current version does not intentionally collect:

- Names.
- Phone numbers.
- Email addresses.
- Payment information.
- Account credentials.
- Chat messages.
- Precise location data.

## Local Storage

The game may use browser local storage to remember progress. Users can clear this data through their browser settings. Clearing browser data may reset local game progress.

## Hosting Provider Logs

The website is hosted by a third-party hosting/CDN provider. The hosting provider may process standard technical logs such as IP address, request path, user agent, and timestamps for security, analytics, debugging, or service operation. Those logs are controlled by the hosting provider's own policies and console settings.

## Future Features

If future versions add accounts, multiplayer, leaderboards, analytics, ads, payments, cloud saves, or social sharing, this policy should be updated before those features are launched.
